# 智能车数据

### 需要改变的数据

- 带宽
- 寻找黄线时的线速度与角速度
- 前进时的线速度
- 转弯时的角速度
- 中线的查找函数



### 数据

初始：

- 带宽=50

- 自旋线速度为0.15

- 自旋角速度为0.86*1.5

- 前进线速度为：

  ```
  linear.x=0.8+0.20×( 1— |bias| ⁄ imagewidth/2)
  ```

  

- 前进角速度为：

  ```
  angular.z=0.02×bias
  ```

### 各个参数的影响：

带宽：

- 过大，转弯不易
- 过小，矫枉过正

自旋线速度：

- 过大，不能转大弯
- 过小，转弯慢
